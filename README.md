## Work description

This project investigates the role of dissipation, entropy, and localization in a quantum reservoir computing (QRC) architecture based on open fermionic systems.

The reservoir is modeled as a network of coupled fermionic modes evolving under the interplay of:

* coherent quantum dynamics (particle hopping and interference),
* disorder/localization effects,
* and dissipative driving induced by an external environment.

Instead of treating dissipation as unwanted noise, the work explores how controlled dissipation can enhance computational properties such as:

* memory capacity,
* stability,
* fading memory,
* and nonlinear response to input signals.

The system is driven by time-dependent inputs through a dissipative target state, generating a nonequilibrium dynamics whose statistical and entropic properties are analyzed.

A systematic parameter sweep is performed over quantities such as:

* disorder strength ((\alpha)),
* dissipation rate ((\gamma)),
* hopping amplitude ((J)),
* graph connectivity ((c)),
* filling fraction,
* and integration timestep ((dt)).

The project studies correlations between:

* reservoir performance (NMSE, memory capacity),
* localization properties,
* and several entropy-based indicators,

with the goal of understanding whether entropic observables can serve as predictors of computational efficiency in open quantum reservoirs.
